# Uppgift 19 - Komprimering

## Läxa
Den här veckan ska ni skriva ett program som komprimerar filer med
hjälp av *Huffmankodning*. Programmet ska kunna både koda (komprimera) och
avkoda (dekomprimera) filer. För att göra det behöver ni generera ett
*Huffmanträd*, det ska genereras från indatafilen och lagras i den komprimerade
filen tillsammans med den huffman-kodade datan. Lägg er programkod i ett repo
med namnet `<kth-id>-compression`.
